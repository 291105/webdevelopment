Het effect op veranderen:

Bij het veranderen naar een inline-block veranderd er niets omdat dit in de originele css al stond.

De inline-block zorgt ervoor dat je inline als block gezien word waardoor de width en height toegepast kon worden.

Bij het veranderen naar block kwamen alle div-elementen onder elkaar te staan. Dit komt omdat bij block je element elke div apart op een regel gezien wordt dus die regel behoort tot deze div. Je height en width worden dus toegepast omdat dit kan bij block elementen.

Bij het veranderen naar een inline element komen alle div elementen naast elkaar te staan. De height en width word genegeerd omdat je dit bij inline niet kan hebben.

Voor welke waarden width en height genegeerd? Dit is bij je inline.

Voor welke waarden elementen als woorden op pagina gezet? Dit is bij je inline omdat dan alle elementen naast elkaar gezet worden en zo een leesbare zin kan vormen.
