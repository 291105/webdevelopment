De tekst staat gecentreerd zodat dit in het midden staat van je rode blok. Doe je dit niet dan zal deze links in je rode blok staan.

Als je in je developer tools kijkt dan zie je dat deze center doorstreept word en dus overschreven wordt. Waarschijnlijk omdat dit automatisch gebeurd in je inline-block.

Vertical aligned bepaald hoe elementen naast elkaar staan dus ten opzichte van elkaar op een regel.

De simpele manier om 1 regel tekst verticaal te centreren is de line-height. Hierbij neem je de waarde van je line-height van het parent element en neem je dezelfde waarde bij je line-height van je child element.

Table-cel wordt vaak gebruikt als truck om meerder regels en dus meerdere blokken verticaal te centreren. Je typt dan de code "display:table-cell".

Waarom krijgt de regel van tall voorrang? Omdat dit eerst in je css werd besproken en css werkt trapgewijs.
